Scenario guides
===============

The Sensu Go Ansible collection covers a wide variety of scenarios, from
installing the software components to managing resources on the running
backends. The documents linked below explain in great detail the specifics of
each supported use case: how to set up an Ansible inventory, what variables to
set, what parameters to use, etc.
